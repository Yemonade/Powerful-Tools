# Powerful-Tools
## 自动清空购物车脚本

`taobao_clear_car.py`

### 环境要求
- python 3.7
- 已安装 `selenium`
### 运行步骤
1. 更改时间服务器为阿里云时间，详情见[此处](https://blog.csdn.net/qq_50357123/article/details/111138072)
2. 运行脚本 `python taobao_clear_car.py`, 按提示输入清空购物车时间
### 尚存问题
流量暴增时无法有效进入目的链接，而无回退功能。
